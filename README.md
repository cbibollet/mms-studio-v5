# 🚤 MMS Studio v5.5 — Mathias Marine Sport

Bienvenue dans l'outil officiel de gestion d'inventaire et de création d'annonces pour **Mathias Marine Sport**. Cet outil web intelligent a été conçu pour optimiser la vente sur Facebook Marketplace.

## 🚀 Fonctionnalités clés

* **📊 Dashboard en temps réel** : Analyse instantanée de la valeur dormante (coût), de la marge potentielle et du nombre d'articles en stock.
* **📦 Gestion d'Inventaire** : Importation directe de ton fichier Excel `VENTE Facebook.xlsx` avec affichage des SKUs, descriptions et alertes pour les stocks de +24 mois.
* **🎨 Studio IA (Détourage)** : Agent expert capable de supprimer automatiquement le fond de tes photos produits pour un rendu professionnel "fond blanc" conforme aux standards Marketplace.
* **📝 Générateur d'Annonces** : Création automatique du texte de vente avec tags optimisés pour une publication rapide.

## 🛠 Installation & Utilisation

1.  Ouvre le lien : [https://cbibollet.github.io/mms-studio-v5/](https://cbibollet.github.io/mms-studio-v5/)
2.  Assure-toi que le voyant **"Expert IA Connecté"** est vert.
3.  Importe ton fichier Excel via l'onglet **Inventaire**.
4.  Prépare tes visuels dans l'onglet **Studio IA**.
